起因

​ 还记得大学语文课上老师讲过的一段话，前面一段大致是老师劝我们好好学习，少打游戏等等。后面老师说：“也许在座的大部分人可能听这些话听腻了，已经麻木了，油盐不进了。但是我还是想让少部分人能听进去，未来少走弯路”；现在想来以前的我真傻，可能现在也傻…生活总是它原本的样子，欠下的迟早都是要还的。

​ 转眼已经毕业两年多了，回想过去两年自己也学了不少东西（如：JavaWeb、数据结构和算法、设计模式、区块链、大数据各个组件等等），但是细想都只停留在’用过’的层次，没有构建自己完整的知识系统。自己也做过很多笔记，但找起来书本都不知道哪去了，所以学着搭建了这个网站，方便记录个人学习笔记、备忘录、灵感、思考、成长。

(花费）时间 -> (踏实)学习 -> (获取)知识 -> (养成)习惯 -> (培养)思维 -> (用心)工作 -> (赚)钱 -> (掌控)生活。

好处

  1. 1次记录，n次阅读，提升复用性。—好记性不如烂笔头
  2. 理清逻辑，培养自己的思维表达写作能力
  3. 可以将知识模块化、建立自己的知识库，持续更新，实现知识的增量积累，查漏补缺转化为存量
  4. 激励自己持续学习和思考，持之以恒，养成习惯
  5. 写博客有讲述的成分，会是一份很好的简历

怎么做

  1. 用手写、用脑子写、用心写。
  2. 辩证的看待问题，没有绝对的正确错误
  3. 记录一些感悟和有价值的东西，不一定非是自己写的，但一定是自己认真斟酌思考过的。—站在巨人的肩膀上看的更远
  4. 无限接近“完成”
  5. 假想有读者，通俗易懂最好。
  6. 扬长避短，就像程序组件被设计出来一样，牺牲一部分来换取另一部分的长。
  7. 多思考，多做阶段性总结，培养兴趣

愿不忘

​ 毕业前老师对我说过的一句话送给各位：“希望你能在工作中努力寻找上升空间，不要给自己设限，不要甘心做一个只会搬砖的码农”。
